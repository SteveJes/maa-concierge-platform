# Club Sportif MAA Source Manifest

This document tracks the approved source inventory for ingestion.

## Source groups

1. **homepage**
   - Captures high-level positioning, key calls to action, and top navigation language.
2. **membership**
   - Membership options and onboarding details.
3. **class_schedule**
   - Class schedule structure and recurring program references.
4. **aquatic**
   - Pool/aquatic programming details and access-related content.
5. **book_a_tour**
   - Tour request pathways and lead-capture flow information.
6. **contact**
   - Contact coordinates and operational contact pathways.
7. **policies (PDFs)**
   - Formal policy documents and structured rule references.

## Why each source group matters

- These groups reflect recurring customer-service intents: understanding offerings, schedule context, aquatic programming, visit booking, and official policy clarification.
- French-first bilingual coverage is maintained by paired `fr-CA` and `en-CA` sources.

## v1 customer-service critical groups

For v1 ingestion priority:
- membership
- class_schedule
- aquatic
- book_a_tour
- contact
- policies

Homepage is included as contextual support, but not a primary answer source for policy-sensitive responses.
